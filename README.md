# What is this?

HAMBoard is a tvOS (Apple TV) application for amateur radio operators.
It combines live DX Cluster monitoring, propagation conditions, contest calendar access, and station-oriented configuration tools in one screen-first dashboard.
This application is dedicated to my uncle UA9XG (SK), who gave me my love for amateur radio.

## Features

- Live DX Cluster spot feed (Telnet)
- Announcements/WWV stream support
- Top bands and top countries rolling statistics overlay
- Propagation pages:
  - Solar conditions summary
  - MUF map
  - foF2 map
- Contest calendar (RSS) with QR link panel
- Configurable DX cluster list (defaults + custom endpoints)
- Raw DX cluster debug stream
